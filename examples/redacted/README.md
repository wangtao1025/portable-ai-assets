# Redacted Examples

This directory is reserved for public-safe walkthrough artifacts such as:
- redacted inspect reports
- redacted diff reports
- sample merge-candidate bundles
- sample reviewed-merge examples
- connector walkthroughs and summary JSON generated from public-safe adapter reports

Rules:
- remove personal names, usernames, chat IDs, and local-only identifiers
- replace machine homes with example paths when publishing
- never include secrets, auth files, or non-redacted memory content
- prefer synthetic examples when real artifacts are too revealing
