# Redacted Project Pack Example

This walkthrough demonstrates the public-safe project pack preview flow:

```bash
./bootstrap/setup/bootstrap-ai-assets.sh --project-pack-preview --both
```

The example project pack contains:

- project instructions;
- curated knowledge-source references;
- read-only action metadata;
- adapter projection source;
- explicit capability risk declarations.

The preview is report-only. It does not upload files, call providers, execute actions, authenticate, or write runtime state.
